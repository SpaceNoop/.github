# 🚀 SpaceNoop

Bienvenue sur l’organisation **SpaceNoop**.

Nous construisons des projets orientés produit, automatisation et développement logiciel avec une approche simple : avancer vite, livrer proprement, et apprendre en continu.

## Ce que vous trouverez ici

- Des projets open source
- Des templates et bonnes pratiques d’équipe
- Des expérimentations techniques
- De la documentation pour contribuer facilement

## Contribuer

Les contributions sont les bienvenues.

1. Ouvrez une issue pour proposer une idée ou signaler un bug
2. Créez une branche dédiée
3. Ouvrez une pull request claire et ciblée

## Contact

Pour toute question, passez par les issues des dépôts concernés.
